1.如何用linux操作系统创建一个c程序

1.首先我们可以先创建一个专属文件夹,然后在该文件夹中创建一个专属c语言文件(由于linux截屏比较麻烦,用手写代码)

$ mkdir first

cd first

touch mkdir first.c

vi first.c

3.进入first.文件,开始编辑c语言

按i进入编辑

int main(){

printf(“hello,world”);

}

4.简单代码编写完成

按esc退出编写模式

按shift+:进入底部命令

输入wq保存内容并退出该.c文件

回到操作系统

5.使用gcc来编译刚才的代码

gcc first.c -o first.exe

此行代码是将编译好的结果用一个新的可执行文件来接受

下次运行代码直接执行该.exe文件即可

即./first.exe

此命令意为执行此目录下的fiest.exe文件

输出结果hello,world

最后可以输入exit以退出linux

2.对进程的了解

1.关于fork函数

1.在父进程中,fork函数创建子进程的PID(进程的ID);

2.在子进程中,fork返回0;

3.未能创建,fork返回负值;

4.我们可以通过fork的返回值判断当前进程;

2.为什么fork要给子进程返回0,给父进程返回子进程的pid?

返回不同的返回值,是为了区分让不同的执行流执行不同的代码块.父进程接收子进程pid是因为父进程可能有多个子进程,为了区分所以用他们特有的pid来作为返回值,而子进程直接用getppid()这个函数就能获得父进程的pid,所以子进程的进程返回就是0;
