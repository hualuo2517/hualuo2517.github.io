创建数据库

查询当前所有已有的数据库

show databases;

新创建一个数据库

create database itcast(命名随意);

删除数据库

drop database itcast;

使用数据库

直接use +数据库名字即可

查询当前所在哪个数据库

select database();

[!NOTE]

1.如果创建的数据库名字有重复,则会报错,如下

2.如果我们要屏蔽错误则可以使用if not exists,如果命名出现问题则不会创建也不会提示错误,当然,数据库中没有任何变化,即没有创建;

3.同理,删除不存在的也可以屏蔽报错;

创建表及表结构

查询当前数据库中所有的表

show tables;

我们先要进入某一数据库中，然后输入命名即可以查看所有的表；

查询某一表的内容

desc(ribe) + 表名；

[!NOTE]

1.如果在编写一个命令时想要强制退出，可以用Ctrl+C；

2.如果想要退出连接，则可以用exit；

3.SQL的默认注释是– ，使用#号也可以，不过没有高亮；
